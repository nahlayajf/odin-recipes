Recipe Webpage

This project is a beginner webpage has different recipes for two beginner meals. It's homepage has the pages to the recipes linked to it.

To complete this project I used HTML to demonstrate skills of:
 - Adding images to a webpage 
 - Creating ordered and unordered list 
 - Using different sized headings
 - Linking different pages on a website to the home page

Additionally, I was able to practice using the my cml and different git commands.
